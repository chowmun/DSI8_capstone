Conclusion & Future improvement

Conclusion

1. Popular filtering is the most straight forward and easy method. However it does not cater for different type of users and preferences.
2. Content based filtering does not need information from other users because it is only depends on items' features. This model will constraint the recommended items to be something similar to user's preference thus less variety.
3. Collaborative based filtering(latent features based matrix factorization) is a better choice for recommendation based on user's preference. However, this model doesn't work well with new user and we could lose some meaningful signals by using a lower-rank matrix.



Future Improvement

1. To incoroperate real time data into the model and constantly update book and user database.
2. Explore the possiblity of recommendation model using the combination of KNN-CF and matrix factorization CF.
